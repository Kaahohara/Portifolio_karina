This portfolio is designed for a creative writer and content creator, developed using JavaScript, CSS, and HTML. The client's request is to infuse vibrant colors and creativity into the design, aiming to make the portfolio more visually appealing and aligned with their creative vision.

The project consists of an online book scheduling system that allows users to reserve books in a virtual library. The system was developed using PHP, SQL, and CSS technologies.

The sistem allows the user to view the available book collection, including information on title, author, and availability.

The implementation of the project involved creating an SQL database to store information about books, users, reservations, and loans. The PHP language was used to implement scheduling functionalities, user registration, login, management of records, and other system functionalities.

In summary, this project offers users a simple and efficient way to make book reservations in a virtual library, while allowing administrators to manage book, user, reservation, and loan records, using PHP, SQL, and CSS technologies to ensure security, quality, and a pleasant user experience.
